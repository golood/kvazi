# kvazi
Приложение для решения задач линейного программирования(ЛП).

### Установить зависимости

```shell
pip install -r requirements.txt
```

### Пример структуры файла с исходными данными
Формат файла: ".json"
```json
{
  "type": "MAX",
  "c": [1, -5, 7],
  "restrictions": [
    [9, -9, 14],
    [2, 7, -8],
    [-2, 5, 4]
  ],
  "comparisonOperators": ["<=", "=", ">="],
  "b": [-9, 13, -1]
}
```
